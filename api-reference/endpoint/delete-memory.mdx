---
title: 'Delete Source'
openapi: 'DELETE /delete_memory'
---

Deletes specific documents or memories associated with a user.

## Sample Request

```bash
curl -X DELETE https://api.usecortex.ai/delete_memory \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "tenant_id": "tenant_456",
    "source_ids": ["doc_123", "doc_456"]
  }'
```

### 🔐 Headers
- `Authorization`: Bearer token

### 🧾 Parameters
- `tenant_id` (string, required): Tenant identifier  
- `source_ids` (list of strings, required): IDs of documents to delete
