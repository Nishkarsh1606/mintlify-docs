---
title: 'List Sources By ID'
openapi: 'POST /list/sources_by_id'
---

Fetches specific sources/documents by their IDs.

## Sample Request

```bash
curl -X POST https://api.usecortex.ai/list/sources_by_id \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "tenant_id": "tenant_456",
    "source_ids": ["doc_123", "doc_456", "doc_789"]
  }'
```

### 🔐 Headers
- `Authorization`: Bearer token

### 🧾 Body Parameters
- `tenant_id` (string, required): Tenant identifier  
- `sub_tenant_id` (string, optional): Optional filter  
- `source_ids` (list of strings, required): IDs of documents
