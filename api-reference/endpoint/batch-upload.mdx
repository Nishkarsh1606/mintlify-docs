---
title: 'Batch Upload'
openapi: 'POST /upload/batch_upload'
---

Uploads multiple documents in a single request for processing and indexing.

## Sample Request

```bash
curl -X POST https://api.usecortex.ai/upload/batch_upload \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "files=@document1.pdf" \
  -F "files=@document2.docx" \
  -F "files=@document3.txt" \
  -F "tenant_id=tenant_456" \
  -F "custom_metadata=[{\"author\":\"John Doe\",\"department\":\"Sales\"},{\"author\":\"Jane Smith\",\"department\":\"Marketing\"},{\"author\":\"Bob Wilson\",\"department\":\"Engineering\"}]"
```

### 🔐 Headers
- `Authorization`: Bearer token

### 🧾 Parameters
- `tenant_id` (string, required): Tenant identifier  
- `sub_tenant_id` (string, optional): Sub-tenant identifier  
- `files` (List of UploadFile, required): Document files  
- `custom_metadata` (stringified JSON list, optional): Metadata list
