---
title: 'Verify Processing'
openapi: 'POST /upload/verify_processing'
---

Checks if a document has been successfully processed.

## Sample Request

```bash
curl -X POST https://api.usecortex.ai/upload/verify_processing \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "file_id": "file_123"
  }'
```

### 🔐 Headers
- `Authorization`: Bearer token

### 🧾 Parameters
- `file_id` (string, required): ID of the uploaded file
